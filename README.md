# 음력 생일 캘린더

## 소개

부모님의 음력 생일을 기억하기 힘들지만 써드파티 캘린더를 사용하기는 싫은 분들을 위한 웹 어플리케이션입니다.

## 사용법

1. 생일을 저장하고 싶은 사람의 이름을 입력합니다.
2. 음력 생일과 저장하고 싶은 햇수를 입력합니다.
3. 이벤트 제목과 설명을 입력합니다.
4. 캘린더에 저장하기 버튼을 클릭!
